# Weather App

> Uma aplicação web simples e intuitiva para consultar as condições climáticas de qualquer cidade do mundo em tempo real.

**[Acesse o projeto online aqui](https://github.com/FilipeLacerda738/app-clima)**

## Sobre o projeto 
Este projeto é uma ferramenta de monitoramento de clima desenvolvida para ser rápida e fácil de usar. O objetivo foi criar uma interface limpa onde o usuário possa obter informações essenciais como temperatura, umidade e vento instantaneamente, tanto no computador quanto no celular.

O design foi pensado para facilitar a leitura dos dados, utilizando um tema escuro e ícones visuais que representam o estado atual do tempo.

## Funcionalidades 

* **Busca Global:** Pesquise pelo nome de qualquer cidade.
* **Dados em Tempo Real:** Informações atualizadas de temperatura atual, descrição do clima (ex: nublado, céu limpo), umidade e velocidade do vento.
* **Design Responsivo:** A interface se adapta automaticamente a telas de smartphones, tablets e desktops.
* **Feedback Visual:** Indicadores de carregamento e mensagens claras caso a cidade não seja encontrada.

## Tecnologias Utilizadas

O projeto foi construído utilizando:

* **HTML e CSS** 
* **JavaScript** 
* **Node.js & Express**
* **OpenWeatherMap API**
